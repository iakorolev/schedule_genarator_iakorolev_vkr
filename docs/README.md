## Поддерживаемые форматы общего расписания

Программа умеет читать два варианта входного файла расписания:

1. Старый формат кафедральной сетки (`ФМиЕН`) с ячейками вида `дисциплина; вид занятия; преподаватель; аудитория`.
2. Новый формат `TDSheet`, который приходит отдельными файлами для бакалавров и магистров, с записями вида `Лекция: ...`, `Лабораторная работа: ...`, `Практические и другие: ...`.

Что поддержано в новом формате:
- автоматический выбор листа `TDSheet`
- извлечение полных ФИО преподавателей
- извлечение аудиторий вида `ОРД - 350`, `ГК - 432`, `ДОТ ДОТ - 000`
- разделение блоков внутри одной ячейки по секциям
- учёт признака недели `В/Н`
- объединённая обработка двух файлов расписания (`sched_bak.xlsx` и `sched_mag.xlsx`) в одной web-сессии
